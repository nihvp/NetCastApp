Get Started

📱 Android Setup

Quick and easy installation:

  1. Go to releases
  2. Download the latest .apk file
  3. Install and enjoy!

🍎 iOS Setup

Run locally with these commands:

git clone https://github.com/nihvp/NetCastApp.git cd NetCastApp npm install cd ios && pod install && cd .. npx react-native run-ios

Make sure you have your React Native environment set up (Node, Watchman, and Xcode).
