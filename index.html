NetCastApp

My LG TV remote disappeared, so I built an open-source, ad-free replacement. Works on iOS and Android.

Download APK Source Code
React Native UDP Networking Zero Tracking AsyncStorage

⚡ Instant Reconnect

Silently re-establishes the connection when you bring the app back from the background. No manual toggling required.

📳 Haptic Feedback

Satisfying tactile responses when you tap buttons, mimicking the feel of a physical remote.

💾 Local Memory

Securely saves your TV's IP address, Session ID, and PIN so you only have to pair it once.

Installation

Android

Grab the latest .apk file from the releases page and install it directly on your device.

iOS

Due to App Store restrictions, you'll need to build it locally:

On your Mac, open Terminal and run:

git clone https://github.com/nihvp/NetCastApp.git
cd NetCastApp && npm install
cd ios && pod install && cd ..
npx react-native run-ios

Make sure you have your React Native environment set up (Node, Watchman, and Xcode).

FAQ

Does this work with older LG TVs?

NetCastApp relies on NetCast protocols. If your TV was made before 2014 and connects to WiFi, it should work seamlessly.

Why do I need to enter a PIN?

This is a standard security feature required by the TV to ensure only authorized devices on your local network can control it.

Is it really ad-free?

Yes. The app is completely open-source and free of ads, tracking scripts, and paywalls.

Built to solve a problem, not sell your data.
